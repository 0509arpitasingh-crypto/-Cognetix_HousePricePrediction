# 🏠 House Price Prediction(Linear Regression)

This project was developed as part of my Machine Learning internship. The objective is to build a model that predicts house prices based on various features such as area, number of bedrooms, bathrooms, and other property-related attributes.

The project covers the complete machine learning workflow, including data preprocessing, exploratory data analysis (EDA), feature selection, model training, and performance evaluation. It provides a practical understanding of how regression algorithms can be applied to solve real-world prediction problems.

## 🚀 Features

* Data cleaning and preprocessing
* Exploratory Data Analysis (EDA)
* Feature selection
* Model training using regression algorithms
* House price prediction
* Model performance evaluation

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn
* Jupyter Notebook

## 📊 Workflow

1. Import and explore the dataset.
2. Clean and preprocess the data.
3. Perform exploratory data analysis.
4. Split the dataset into training and testing sets.
5. Train the machine learning model.
6. Evaluate the model using appropriate metrics.
7. Predict house prices on unseen data.
